# TranscriptX Roadmap

## Now (stabilising)
- Core CLI contract
- Output determinism
- Module interface consistency
- Keep existing local WebUI working (no new dashboard features)

## Next (high leverage)
- Plugin/adapter abstraction (Slack, Telegram, etc.)
- Lightweight UI layer
- Better stats and summaries

## Non-goals
- SaaS
- Black-box AI
- Cloud-only workflows
